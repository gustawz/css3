Desenvolvimento Web é termo utilizado para descrever o desenvolvimento de sites, na Internet ou numa Intranet



O profissional que trabalha desenvolvendo websites pode ser um Web Designer (Desnvolvedor de Loyout), ou Web Developer (desenvolvedor de Sistemas).



O desenvolvedor refere-se a um processo de construção de testes de software específico para a Web, com a dinalidade de se obter um conjunto de programas que satisfazem as funções pretendidos, quer em termosde usabilidade dos usuários ou compatibilidade com outros programas existentes.



O Desenvolvimento Web pode variar desde simples páginas estáticas a aplicações ricas, comércios aletrônicos ou redes sociais
